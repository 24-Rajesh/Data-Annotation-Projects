# Data Annotation Projects

This repository contains beginner data annotation projects created using Microsoft Excel to demonstrate labeling, classification, and LLM evaluation workflows.

## Projects Included

### 1. LLM Prompt Response Quality Review

Reviewed AI-generated responses and labeled outputs based on helpfulness, relevance, accuracy, and safety.

### 2. User Query Intent Classification

Classified user messages into categories such as Question, Complaint, Request, Feedback, and Greeting.

## Tools Used

* Microsoft Excel
* GitHub

## Purpose

To showcase practical annotation skills relevant to AI, LLM, and human-in-the-loop workflows.
